## Quick start

First of all, install [Leiningen](https://leiningen.org/)
and [Node](https://nodejs.org/en/), then follow these steps:

### Installing dependencies

```bash
lein deps
```

### Compiling & preparing

```bash
lein build
```

### Running the user interface

```bash
lein start-ui
```
